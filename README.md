# Le Pavillon — Ben & Fitness

Application de coaching personnalisé pour 30 clients.

## Stack
- **Frontend** : Next.js (React)
- **Base de données** : Supabase (PostgreSQL)
- **Auth** : Supabase Auth
- **Hébergement** : Vercel (gratuit)

## Modules
- 🏠 Dashboard client avec stats & message du coach
- 🏋️ Entraînements — programme semaine + suivi des séances
- 🥗 Nutrition — plan alimentaire + macros
- 🎯 Objectifs & progression — mesures + records personnels
- 💬 Messages — chat coach-client en temps réel
- 👨‍💼 Panel coach — gestion de tous les clients

## Déploiement
Voir `GUIDE-DEPLOIEMENT.md` pour les instructions complètes.
